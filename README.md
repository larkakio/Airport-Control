# Airport Control

Neon mobile air-traffic game + daily **check-in** on [Base](https://base.org), built with Next.js (`web/`) and Foundry (`contracts/`).

- **Live app:** [airport-control.vercel.app](https://airport-control.vercel.app)
- **Repo layout:** deploy the `web/` folder on Vercel; set env vars from `web/.env.example`.
